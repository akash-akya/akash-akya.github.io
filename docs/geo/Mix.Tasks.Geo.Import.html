    View Source mix geo.import (geo v0.1.0)

    Import from IP location data from CSV file.

    CSV file must contain first line as header for the column. Columns can appear in any order and optional fields can be omitted.

    Column names should be ip_address, country_code, country, city, latitude, longitude, mystery_value

    all additional columns are ignored.

    examples

     Examples 

    $ mix geo.import sample.csv
$ mix geo.import sample.csv --verbose

    command-line-options

     Command line options

    • --verbose - log discarded rows
